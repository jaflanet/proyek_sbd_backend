pada file backend ini terdapat beberapa komponen yang dibagi menjadi part-part
1. file server.js merupakan file root dari back end ini
2. file pool untuk menghubungkan dengan postgres
3. file queries agar dapat mengirimkan queries 
4. terdapat folder controller yang berisi queries-queries yang akan dikirimkan
5. folder routes untuk menghubungkan request dari API ke database
